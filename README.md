# World Cup 2026 Personal Fixture Hub ⚽🏆

All 104 FIFA World Cup 2026 matches shown in **Dhaka time (GMT+6)** — with search,
favourite teams, countdowns, group tables, a knockout bracket, and auto-updating scores.

**🔗 Live app:** https://worldcup-hub-sigma.vercel.app/

## Features
- Every fixture converted to Dhaka time
- Search by team, group, city or date + quick filters
- Favourite teams saved in your browser
- Live countdowns and Upcoming / Live / Finished status
- Group standings that calculate from results
- Knockout bracket + dark/light theme

## Tech
Plain HTML, CSS and JavaScript (no frameworks). Live scores via the free
football-data.org API, fetched through a Vercel serverless function so the key stays private.
